特别关注：

  • • 禁止在玩具中故意使用全氟烷基和多氟烷基物质(PFAS)

  • • 36个月以下儿童使用的玩具和设计为放入口中的玩具完全禁止使用任何香精

新增化学物质的专项管控

EN71-15:2025

发泡玩具材料中的甲酰胺

  • • 甲酰胺含量≤200mg/kg
  • • 适用范围：拼图地垫、玩具沙发、填充玩具等
  • • 超标需遵守玩具安全指令附录中规定的排放限值

EN71-16:2025

玩具材料中的氯化磷阻燃剂

  • • TCEP、TCPP、TDCP每种物质均≤5mg/kg
  • • 适用范围：塑料部件、涂层、纺织品、电子元件绝缘材料等

EN71-17:2025

水性玩具材料中的异噻唑啉酮类防腐剂

  • • MIT≤0.25mg/kg，CIT≤0.75mg/kg，BIT≤5.0mg/kg
  • • 适用范围：手指画颜料、黏土、泡泡水等

* 这三项标准将于2025年8月底前在各成员国实施，与现行国家标准冲突的部分将被取代

物理安全要求的强化与细化

小零件和锐利边缘管控

  • • 36个月以下儿童玩具滥用测试后，不得产生直径≤31.7mm的部件
  • • 所有可触及边缘半径需≥0.5mm
  • • 金属边缘厚度需≥0.5mm

拉力、扭力和跌落测试

  • • 附着小部件需承受90N力持续10秒不脱落
  • • 36个月以下儿童玩具部件需经20N拉力测试不脱落
  • • 毛绒玩具缝合处需承受≥70N拉力
  • • 跌落测试：85cm高度，重复3-5次

磁性部件严格限制

  • • 磁通量指数≥50kG²mm²的磁性部件纳入高风险管控
  • • 能完全放入小零件测试筒的磁铁，磁通量指数必须<50kG²mm²
  • • 两个磁体间距≤50mm时吸附力超过0.5N即判定为不合格

玩具类型 儿童年龄 连续噪音限值 脉冲噪音限值
近耳玩具 <18个月 ≤60 dB(A) ≤75 dB(C)
近耳玩具 18-36个月 ≤65 dB(A) ≤85 dB(C)

手持/桌面玩具 - - ≤115 dB(C)

部分近耳玩具的连续噪音限值从80dB降低到70dB，C加权峰值声压从115dB降低到110dB

数字产品护照(DPP)的全面推行

技术要求

  • • 每个玩具必须配备唯一的数字标识（如二维码或RFID标签）
  • • 消费者无需下载软件、注册或输入密码即可访问
  • • 在线销售时，必须允许买家在购买完成前看到指向DPP的链接

内容要求

  • • 合规性证明：EC符合性声明、CE标志的数字版本
  • • 产品溯源信息：制造商名称和地址、生产批次号、生产日期
  • • 材料成分清单：精确到每种塑料、涂料、纺织品
  • • 化学安全测试报告：REACH、POPs、PAHs等测试结果
  • • 安全警告和使用说明：多语言版本

监管价值

  • • 市场监管和海关检查更加简便高效
  • • 实现玩具安全监管的数字化转型
  • • 监管机构可实时验证产品合规性
  • • 快速识别和处理不合格产品

实施时间表：

欧盟委员会将在TSR生效后12个月内通过授权法案，明确DPP相关技术要求，并发布实施指南及定制化指引

CE认证流程的新变化

风险分级管理

  • • 高风险玩具（含电子元件、化学材料、小部件等）需100%验货或验厂
  • • 低风险产品（如部分毛绒玩具）可简化流程
  • • 2025年起，取消传统的"一次性模式"与"注册模式"

公告机构强制介入

  • • 涉及电子支付、网络连接、高压组件或复杂机械结构的产品，强制需由欧盟公告机构发证
  • • 2025年8月1日起，涉及网络连接、个人数据处理的玩具需额外符合EN 18031网络安全标准
  • • 若制造商未采用欧盟官方公报上发布的完全协调标准，必须由欧盟公告机构参与符合性评定

认证标准更新

  • • 电子玩具需符合更新版EN62115标准，新增电磁场（EMF）风险评估要求
  • • 新增有机化合物迁移测试（EN71-9），涵盖19种有害物质
  • • 填充物需通过EN71-12微生物测试，防止霉菌滋生

数字玩具的心理健康评估要求

新规在玩具安全评估中首次引入了心理健康和认知发展风险评估，这是玩具安全监管史上的重大创新。

心理风险评估

  • • 游戏沉迷机制评估：评估玩具是否存在诱导儿童过度沉迷的设计元素
  • • 内容适宜性评估：评估玩具内容对不同年龄段儿童的适宜性
  • • 社交互动影响：评估玩具的社交功能对儿童人际交往能力的影响

认知发展评估

  • • 学习效果评估：评估玩具是否真正促进儿童认知发展
  • • 注意力影响：评估玩具对儿童注意力发展的影响
  • • 思维模式塑造：评估玩具可能塑造的思维模式

人工智能玩具特殊要求

根据欧盟法规(EU)2024/1689，将人工智能系统作为安全组件且需经第三方符合性评定的玩具，应归类为高风险人工智能系统。

卖家注册商品的平台审核新要求

电商平台的合规义务升级

  • • 平台必须设计其系统，允许卖家在购买完成前显示CE标志、安全警告和指向DPP的链接
  • • 所有在线销售的玩具必须带有清晰可见的DPP，显示符合相关安全规则
  • • 平台必须明确显示所有列出玩具的警告、标记和DPP链接
  • • 平台需要在欧盟安全门户（Safety Gate Portal）注册，标明单一联络点信息

文档一致性要求的严格化

核心一致性要求：

  • • 产品名称和型号：必须在测试报告、CE证书和物理产品标签上完全一致
  • • 避免使用拼音或缩写，确保所有文档使用统一的标准名称
  • • 制造商信息：名称、地址、联系方式等必须一致
  • • 认证信息：CE标志、认证机构编号等必须准确无误

文档上传和验证的新流程

亚马逊"直接验证制度"流程：

  1. 卖家在"管理您的合规性（MYC）"页面提交测试请求
  2. 指定TIC机构收到请求后与卖家联系，安排检测或验证已有报告
  3. 勾选需要检测的ASIN，选择一个TIC机构
  4. 选择寄送样品的实验室地址
  5. 与TIC联系，确认报价，寄送样品
  6. 等待检测完成，结果将自动上传到亚马逊

审核时间：约5-7个工作日

违规处理机制

产品处理：

  • • 未按规定提交合格报告的商品将被禁止销售
  • • 不合规产品将被直接下架
  • • 产品将被移除销售权限，无法正常销售

卖家处罚：

  • • 可能面临封店风险
  • • 多次违规将导致账号限制或封禁
  • • 需承担因产品问题造成的法律责任和经济损失

不同类型玩具的差异化要求

36个月以下婴幼儿玩具

  • • 完全禁止使用任何香精
  • • 滥用测试后不得产生直径≤31.7mm的部件
  • • 小部件拉力测试要求更严格，需承受20N拉力不脱落
  • • 重金属迁移限值最为严格，如铅从13.5mg/kg降至2.0mg/kg
  • • 必须明确标注"0-3岁"使用年龄

电动和智能玩具

  • • 需符合更新版EN62115标准
  • • 工作电压≤24V DC/AC
  • • 电池过热保护：130℃自动断电
  • • 新增电磁场（EMF）风险评估要求
  • • 2025年8月1日起，需符合EN 18031网络安全标准

磁性玩具

  • • 磁通量指数≥50kG²mm²的磁性部件纳入高风险管控
  • • 能完全放入小零件测试筒的磁铁，磁通量指数必须<50kG²mm²
  • • 两个磁体间距≤50mm时吸附力超过0.5N即判定为不合格
  • • 需评估肠道梗阻风险

平台应对策略与实施建议

系统改造的优先级规划

第一阶段（立即执行）：

  • • DPP技术对接：开发支持DPP二维码扫描和信息展示的系统功能
  • • 文档验证系统：升级文档验证系统，实现对CE证书、测试报告等文件的自动校验
  • • 语言支持：确保平台支持英/德/法语的技术文件展示

第二阶段（3个月内完成）：

  • • 风险分级系统：建立玩具产品风险分级管理系统
  • • 一致性校验：开发文档一致性自动校验功能
  • • TIC机构对接：与认可的TIC机构建立API接口

审核流程的标准化设计

自动审核（第一级）：

  • • 文档完整性检查：确保所有必需文档已提交
  • • 格式验证：检查文档格式是否符合要求
  • • 一致性校验：自动比对产品名称、型号、制造商信息等关键数据
  • • 有效期检查：验证证书和测试报告的有效性

人工审核（第三级）：

  • • 复杂案例处理：针对系统无法自动判断的复杂情况
  • • 风险评估：对高风险产品进行深入的合规性评估
  • • 投诉处理：处理消费者投诉和监管部门的问询

结语
